Bien démarrer

Logiciels utiles

Pour travailler en LaTeX, il vous faut deux éléments : une distribution TeX, et un éditeur de texte orienté LaTeX.

Distribution

  • Pour windows : choisissez MiKTeX ou TeX Live. Si vous choisissez MiKTeX, installez aussi GhostScript (mais vous n'avez pas besoin de GhostView).
  • Pour Mac OS X, installez MacTeX, et éventuellement les MacTeXtras.
  • Pour Linux, installez TeX Live, soit la version nature soit celle de votre distribution Linux. Si plusieurs paquets sont disponibles, installez au moins ceux dont le nom contient latex ou recommended (ou installez-les tous).

Éditeur

  • Pour windows : TeXworks (inclus dans TeX Live), TeXnicCenter, TeXmaker ou LeD (pas libre) voire WinEDT (shareware).
  • Pour Mac OS X : TeXworks, TeXshop (tous deux dans MacTeX), iTeXmac, TeXmaker ou Aquamacs (tous trois dans MacTeXtras) ou TextMate (shareware).
  • Pour Linux : TeXworks, Kile ou TeXmaker.

Source minimal

LaTeX ayant été conçu par un Américain à l'époque où on ne parlait pas beaucoup d'internationalisation, quelques ajustements sont nécessaires pour taper un texte en français. Un source minimal pour le français ressemble à :

   \documentclass[a4paper]{article} % papier A4
   \usepackage[utf8]{inputenc}      % accents dans le source
   \usepackage[T1]{fontenc}         % accents dans le pdf
   \usepackage{textcomp}            % symboles complémentaires (euro)
   \usepackage[frenchb]{babel}      % titres en français
   \begin{document}
   Bonjour, monde accentué ! Ça va ?
   \end{document}

Selon votre éditeur de texte et la façon dont il est réglé, il vous faudra changer utf8 en latin1 ou applemac à la ligne 2. À l'inverse, vous pouvez (sauf avec certains éditeurs sous windows) régler votre éditeur sur utf8 via son menu outils ou préférences.

Références utiles

Livres

Les deux meilleurs livres en français pour apprendre LaTeX sont actuellement :

  • LaTeX pour l'impatient, C. Chevalier et W. Appel, éditions H&K (amazon)
  • LaTeX, cours et exercices, D. Bitouzé et J.-C. Charpentier, éditions Pearson (amazon)

Deux références classiques mais un peu vieillies sont les livres de B. Desgraupes et C. Rolland. Une fois assimilés les livres ci-dessus, le livre de référence est le

  • LaTeX Companion, F. Mittelbach et M. Goossens, Pearson (amazon)

Documents en ligne

  • Une courte (?) introduction à LaTeX par T. Oetiker, trad. S. Colin et votre serviteur.
  • Mon polycopié de cours de licence 2 sur LaTeX, et ses fichiers d'accompagnment.
  • Les supports de cours et exercices de Philippe Goutet pour le même cours l'année suivante (plus complets, avec une présentation de TikZ notamment).

Foires aux questions

Deux FAQ francophones sont disponibles : une ancienne au format html et une moins ancienne en pdf.

La FAQ anglophone reste la plus à jour (utiliser la case google pour vos recherches). Elle est accompagnée d'un guide visuel bien pratique.

Aide humaine en ligne

Plusieurs moyens existent pour obtenir de l'aide en ligne sur LaTeX. Les plus recommandés en français sont :

  • le groupe usenet fr.comp.text.tex, accessible avec un lecteur de nouvelles ou via Google Groups ;
  • la liste de diffusion de GUTenberg, par mail après abonnement ;
  • les sections LaTeX du forum web Mathematex.

En anglais, les principaux lieux pour obtenir de l'aide sont :

  • le groupe usenet comp.text.tex ;
  • la mailing-list texhax ;
  • le site TeX du réseau StackExchange.

Groupes d'utilisateurs

Si vous devenez un utilisateur régulier de LaTeX, vous pouvez vouloir rejoindre un groupe d'utilisateur pour bénéficier des avantages (DVDs de la collection TeX, publications) mais aussi soutenir le développement de TeX.

  • Groupe francophone d'utilisateurs : GUTenberg.
  • Groupe international : le TUG.
