Tous les fichiers — LM204

Les types de fichiers sont les suivants (où NN est le numéro de la séance) :

  • cours.pdf : le polycopié de cours ;
  • NN-<nom>.pdf : support de cours, NN-<nom>-3up.pdf : pour imprimer ;
  • exos-NN.zip : énoncés, solutions, corrigés, fichiers annexes  pour la séance NN, voir les détails en fin de chapitre 1 du cours ;
  • doc-<description> : document annexe.
  • tout-XXX.zip : regroupement d'autres fichiers ;
  • sources.zip : les sources du cours, des exercices et des documents.
